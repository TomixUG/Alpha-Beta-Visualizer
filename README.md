# Alpha-Beta Visualizer

Browser tool to visualize the alpha-beta pruning optimization technique for the minimax algorithm. [Check it out here!](https://alpha-beta-visualizer.netlify.app/)

Forked from [this repository](https://github.com/BSmith156/Alpha-Beta-Visualizer). Added support for positive and negative infinity in static eval field.
